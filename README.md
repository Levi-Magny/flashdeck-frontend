# Flashdeck Frontend

## Project Description
Flashdeck is an innovative platform designed to help users effectively learn and manage their flashcards for studying various subjects. It provides a clean interface with efficient tools for creating, reviewing, and tracking flashcards to enhance learning.

## Tech Stack
- **Frontend:** React.js
- **State Management:** Redux
- **Styling:** CSS Modules
- **Testing:** Jest, React Testing Library

## Features
- User authentication (sign up, log in, log out)
- Create, edit, and delete flashcards
- Support for categories and tags
- Progress tracking and analytics
- Dark mode and light mode themes

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Levi-Magny/flashdeck-frontend.git
   cd flashdeck-frontend
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
4. Open your browser and navigate to `http://localhost:3000` to see the app in action.
